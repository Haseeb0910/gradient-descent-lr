# 📈 Linear Regression & Gradient Descent

An interactive visual tool that shows exactly how a machine learning model learns — 
built from scratch using only Python and NumPy, with a Streamlit UI.

## 🚀 Live Demo
👉 [gradient-descent-lr.streamlit.app](https://gradient-descent-lr.streamlit.app)

## 🎯 What It Does
Adjust sliders for learning rate, epochs, noise, and sample size — 
and watch the model retrain in real time with 4 live visualizations.

## 📊 Visualizations
- **Loss Convergence Curve** — tracks how error drops each epoch
- **Regression Fit Progression** — shows the line fitting over epochs 0 → 100
- **2D Cost Contour Map** — gradient descent path in parameter space
- **3D Cost Surface** — full optimization landscape with GD trajectory

## 🧠 Built From Scratch (No sklearn)
- Mean Squared Error (MSE) cost function
- Batch Gradient Descent
- Manual weight & bias parameter updates

## 🛠️ Tech Stack
Python · NumPy · Matplotlib · Streamlit

## ▶️ Run Locally
```bash
git clone https://github.com/Haseeb0910/gradient-descent-lr.git
cd gradient-descent-lr
pip install -r requirements.txt
streamlit run app.py
```
